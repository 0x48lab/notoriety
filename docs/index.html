Overview

Notoriety is a reputation system inspired by classic MMORPGs. Your actions determine your standing - good players are protected, while criminals face consequences.

Key Features

  • Name Color System - Blue (Innocent), Gray (Criminal), Red (Murderer)
  • Alignment System - -1000 to +1000 good/evil axis
  • PK Tracking - Killing innocent players has lasting consequences
  • Property Protection - Blue players' property is protected
  • Bounty System - Place bounties on murderers
  • Village Integration - Villagers and golems react to your reputation

Parameters

Parameter Range Initial Purpose
Alignment -1000 ~ +1000 0 Good/Evil axis (negative = gray)
PKCount 0 ~ unlimited 0 Murder tracking (1+ = red)
Fame 0 ~ 1000 0 Title determination

Name Color System

Your name color reflects your reputation status and determines how NPCs and other systems treat you.

Blue (Innocent)

Alignment ≥ 0, PKCount = 0

  • Trade with villagers at normal prices
  • Your property is protected
  • Keep items on death
  • Golems are neutral
  • High alignment = villager discounts

Gray (Criminal)

Alignment < 0, PKCount = 0

  • Villager prices are 1.5x higher
  • No ownership on new placements
  • No penalty for being attacked
  • Keep items on death
  • Golems attack if caught committing crimes

Red (Murderer)

PKCount ≥ 1

  • Villagers refuse to trade
  • Golems attack on sight
  • Bounty can be placed on you
  • Drop items on death
  • No ownership on new placements

State Transitions

Blue → Gray Alignment drops below 0
Gray → Blue Alignment reaches 0 or above
Blue/Gray → Red Kill a blue player (PKCount +1)
Red → Gray PKCount reaches 0 (time decay)
Note: You cannot go directly from red to blue. You must first become gray and restore your alignment.

Crimes and Penalties

Committing crimes decreases your Alignment. Penalties are tiered into three levels:

Severe Crimes (-50)

Action Penalty Notes
Kill another player -50 PKCount +1
Kill villager (with bed) -50 Villager linked to a bed
Steal from player's container -50 Taking items from protected chests
Kill Iron Golem -50 Nearby golems become hostile

Moderate Crimes (-10)

Action Penalty Notes
Kill animals -10 Gray or above. When witnessed by villager or golem
Kill villager (no bed) -10 Villager not linked to a bed

Petty Crimes (-5)

Action Penalty Notes
Destroy player's property -5 Replace within 5 sec to cancel
Destroy villager's bed -5 Exempt if you or trusted player placed it
Destroy villager's workstation -5 Exempt if you or trusted player placed it

Trivial Offenses (-1)

Action Penalty Notes
Attack another player -1 Attacking blue players
Attack player's pet -1 Blue player's tamed animals
Attack villager -1 Even without killing
Harvest crops -1 Gray or above. When witnessed by villager
Tip: Blocks placed by you or your trusted players can be freely modified without penalty.

Recovery and Good Deeds

Alignment naturally recovers toward 0 over time, and good deeds speed up the process.

Action Recovery Notes
Time passage +10/hour Moves alignment toward 0
Kill red player +50 Bounty hunting
Trade with villager +5 Each successful trade
Kill monster +1 Fame also increases

PKCount Decay

When Alignment reaches 0 while red: PKCount -1, Alignment resets to -1000

This means murderers must spend significant time recovering before returning to innocent status.

Bounty System

Place bounties on red players to incentivize their capture.

Placing Bounties

  • Target: Red players only
  • Minimum: 100 currency
  • Stacking: Multiple bounties on same player add up

Claiming Bounties

Reward Type Amount
Bounty Money Total pooled bounty
System XP (PK 1-4) 50 XP
System XP (PK 5-9) 100 XP
System XP (PK 10-19) 300 XP
System XP (PK 20+) 500 XP
Fame +50, or inherit target's Fame if higher

Bounty Signs

OPs can create bounty ranking signs by placing a sign with [bounty] on the first line.

Trust System

Trust players to share property access without committing crimes. Uses a three-state system.

Three-State Trust

State Description Priority
TRUST Explicitly trusted - full access allowed High
UNSET No setting - falls back to guild trust Medium
DISTRUST Explicitly distrusted - access denied even for guild members Highest

Effects of Trust

  • Open and take from trusted player's chests
  • Break trusted player's blocks
  • Attack trusted player without penalty
  • Kill trusted player without PK count

Distrust Effects

  • Can open chests but cannot take items
  • Can put items into chests
  • Overrides guild membership trust
One-Way Trust: Trust is not mutual. If you trust someone, they can access your property, but you cannot access theirs unless they also trust you.

Guild System

Form guilds to automatically share trust with all members. Guild members can access each other's property without explicit trust settings.

Key Features

  • Creation Cost - 50,000 gold
  • Automatic Trust - Guild members automatically trust each other (unless explicitly distrusted)
  • Guild Tags - 1-4 character tags displayed before player names
  • Customizable Colors - 16 colors available for guild tags
  • Role System - Master, Vice Master, Member roles
  • Invitation System - 7-day invitation expiry

Roles

Role Permissions
Master All permissions - disband, kick, promote, demote, invite, tag color
Vice Master Invite new members
Member Access guild features, leave

Trust Priority

1. DISTRUST → Access denied (highest priority)

2. TRUST → Access allowed

3. Same Guild → Access allowed (if UNSET)

4. Otherwise → Access denied

Guild Application

Players can apply to join a guild without waiting for an invitation.

  • /guild apply <guild_name> - Apply to join a guild
  • /guild applications - View pending applications (Master/Vice Master)
  • /guild applications accept <player> - Accept an application
  • /guild applications reject <player> - Reject an application

Application expiry: 7 days

Player Display

Guild members have their guild tag displayed after their name.

Format: Title PlayerName [GuildTag]

Example: Lord PlayerName [BC]

Territory System

Guilds can claim protected territories. Territory size scales with member count.

Basic Info

Item Details
Requirement Minimum 5 guild members
Size 1 chunk per 5 members
Beacon Auto-placed at territory center

Territory Protection

Action Guild Member Non-Member
Place blocks ✅ Allowed ❌ Denied
Break blocks ✅ Allowed ❌ Denied
Chest access ✅ Allowed ⚠ Theft is crime
Movement ✅ Allowed ✅ Allowed

Entry Notifications

Players receive messages when entering or leaving territories.

  • ▶ Entering [GuildName]'s territory
  • ▶ Entering your guild's territory
  • ◀ Leaving [GuildName]'s territory

Territory Commands

Command Description
/guild territory set Set current location as territory center (Master only)
/guild territory info View territory information
/guild territory remove Remove territory (Master only)
Guild Master Login: Guild masters receive territory status notifications on login (chunks used, expansion available).

Chat System

Multiple chat channels for different communication needs.

Chat Channels

Channel Range Prefix Command
Local 50 blocks (none) /chat local
Global Server-wide ! /chat global
Guild Guild members only @ /chat guild
Whisper 1-to-1 - /w <player>

Chat Display Format

  • Local: [TAG] PlayerName: message
  • Global: [G][TAG] PlayerName: message
  • Guild: [Guild] PlayerName: message
  • Whisper: [PlayerName → You]: message

Romaji Conversion

Auto-converts romaji input to hiragana for Japanese players.

  • /chat romaji on - Enable conversion
  • /chat romaji off - Disable conversion
  • URLs, numbers, and commands are not converted

Inspect System

Investigate block ownership and territory information.

Inspection Methods

  • Inspect Mode - Toggle with /inspect. While ON, clicking blocks shows their info
  • Inspect Stick - Get with /inspect tool. Click blocks with this item to inspect

Displayed Information

Info Description
Block Type Type of the clicked block
Location World name and coordinates
Owner Player who placed the block
Placed At When the block was placed
Trust Status Your trust relationship with the owner
Territory If in territory, shows guild name
