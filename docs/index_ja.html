概要

Notorietyは、クラシックMMORPGにインスパイアされた名声システムです。あなたの行動が評価を決定し、善良なプレイヤーは保護され、犯罪者は制裁を受けます。

主な機能

  • ネームカラーシステム - 青（Innocent）、灰（Criminal）、赤（Murderer）
  • Alignmentシステム - -1000〜+1000の善悪軸
  • PK追跡 - 無実のプレイヤーを殺すと永続的な影響
  • 財産保護 - 青プレイヤーの所有物は保護される
  • 懸賞金システム - 殺人者に懸賞金をかけられる
  • 村との連携 - 村人とゴーレムがあなたの評判に反応

パラメータ

パラメータ 範囲 初期値 用途
Alignment -1000〜+1000 0 善悪軸（負=灰色）
PKCount 0〜上限なし 0 殺人追跡（1以上=赤）
Fame 0〜1000 0 称号の決定

ネームカラーシステム

あなたの名前の色は評判を反映し、NPCや各種システムがあなたをどう扱うかを決定します。

青（Innocent）

Alignment ≥ 0、PKCount = 0

  • 村人と通常価格で取引可能
  • 所有物が保護される
  • 死亡時にアイテムを保持
  • ゴーレムは中立
  • 高Alignmentで村人から割引

灰（Criminal）

Alignment < 0、PKCount = 0

  • 村人取引が1.5倍の価格
  • 新規設置物に所有権なし
  • 攻撃されてもペナルティなし
  • 死亡時はアイテム保持
  • 犯罪を目撃されるとゴーレムが攻撃

赤（Murderer）

PKCount ≥ 1

  • 村人が取引を拒否
  • ゴーレムが視認で攻撃
  • 懸賞金をかけられる
  • 死亡時にアイテムを落とす
  • 新規設置物に所有権なし

状態遷移

青 → 灰 Alignmentが0未満になる
灰 → 青 Alignmentが0以上になる
青/灰 → 赤 青プレイヤーを殺害（PKCount +1）
赤 → 灰 PKCountが0になる（時間経過）
注意: 赤から直接青には戻れません。まず灰色になり、Alignmentを回復する必要があります。

犯罪とペナルティ

犯罪を行うとAlignmentが減少します。ペナルティは3段階に分類されています：

重大犯罪（-50）

行動 ペナルティ 備考
村人を殺害（ベッドあり） -50 ベッドに紐づいている村人
他プレイヤーの所有物を盗む -50 保護されたチェストからアイテムを取る
アイアンゴーレムを殺害 -50 周囲のゴーレムも敵対化

中程度の犯罪（-10）

行動 ペナルティ 備考
他プレイヤーの所有物を破壊 -10 5秒以内に戻せばキャンセル
村人を殺害（ベッドなし） -10 ベッドに紐づいていない村人

中程度の犯罪（-20）

行動 ペナルティ 備考
動物を殺す -20 村人またはゴーレムに目撃された場合

軽微な犯罪（-1）

行動 ペナルティ 備考
作物を収穫する -1 村人に目撃された場合のみ
他プレイヤーを攻撃 -1 青プレイヤーへの攻撃
他プレイヤーのペットを攻撃 -1 青プレイヤーの飼いならした動物
村人を攻撃 -1 殺害しなくても
村人のベッドを破壊 -5 自分または信頼者が設置した場合は免除
村人の仕事場を破壊 -5 自分または信頼者が設置した場合は免除
ヒント: 自分で設置したブロックや、信頼しているプレイヤーが設置したブロックはペナルティなしで操作できます。

回復と善行

Alignmentは時間経過で自然に0に向かって回復し、善行で回復を早めることができます。

行動 回復量 備考
時間経過 +10/時間 Alignmentが0に向かって移動
赤プレイヤーを討伐 +50 賞金稼ぎ
村人と取引 +5 取引成功ごと
モンスターを討伐 +1 Fameも増加

PKCount減少

赤プレイヤーのAlignmentが0に達すると：PKCount -1、Alignment = -1000にリセット

つまり、殺人者が無実に戻るには長期間の回復が必要です。

村との連携

村人とアイアンゴーレムはあなたの評判に反応します。

村人との取引

状態 価格倍率
青（高Alignment 500+） 5〜20%割引
青（通常） 通常価格
灰 1.5倍
赤 拒否

アイアンゴーレムの挙動

状態 ゴーレムの反応
青（高Alignment） 護衛としてモンスターから守る
青（通常） 中立
灰 犯罪を目撃すると攻撃（32ブロック範囲）
赤 視認で攻撃（128ブロック検索範囲）
強化ゴーレム: ゴーレムが犯罪者を攻撃する際、速度とダメージが強化されます。遠距離のターゲットにはテレポートします。

懸賞金システム

赤プレイヤーに懸賞金をかけて、捕縛を促進できます。

懸賞金をかける

  • 対象: 赤プレイヤーのみ
  • 最低金額: 100
  • 累積: 複数人が同じプレイヤーにかけると合算

懸賞金の獲得

報酬タイプ 金額
懸賞金 累積された全懸賞金
システム経験値（PK 1-4） 50 XP
システム経験値（PK 5-9） 100 XP
システム経験値（PK 10-19） 300 XP
システム経験値（PK 20+） 500 XP
Fame +50、または相手のFameが高ければ継承

懸賞金看板

OPは看板の1行目に[bounty]と入力することで懸賞金ランキング看板を作成できます。

信頼システム

プレイヤーを信頼することで、犯罪にならずに財産を共有できます。

三段階信頼システム

信頼状態は「信頼」「未設定」「不信頼」の三段階で管理されます。

状態 効果 コマンド
TRUST（信頼） 所有物へのフルアクセスを許可 /trust add <player>
UNSET（未設定） デフォルト。ギルドメンバーなら自動信頼 /trust remove <player>
DISTRUST（不信頼） ギルドメンバーでもアクセス拒否 /trust distrust <player>

信頼の効果

  • 信頼したプレイヤーのチェストを開いてアイテムを取れる
  • 信頼したプレイヤーのブロックを破壊できる
  • 信頼したプレイヤーを攻撃してもペナルティなし
  • 信頼したプレイヤーを殺してもPKカウント増加なし

不信頼時のチェスト動作

  • チェストを開くことは可能（中身を見れる）
  • アイテムを取り出すことは不可
  • アイテムを入れることは可能（所有者への利益）
片方向の信頼: 信頼は相互ではありません。あなたが誰かを信頼しても、相手があなたを信頼しない限り、あなたは相手の財産にアクセスできません。

ギルドシステム

ギルドは信頼関係をグループ化する機能です。同じギルドのメンバーは自動的に信頼扱いになります。

主な機能

  • 作成費用 - 50,000ゴールド
  • 自動信頼 - ギルドメンバー間は自動的に信頼関係が構築される
  • ギルドタグ - 名前の前に表示される1-4文字のタグ（例: [BC] PlayerName）
  • ギルドチャット - @プレフィックスでメンバーのみに送信
  • 役職システム - マスター、副マスター、メンバーの3役職

役職と権限

役職 権限
マスター 全権限（解散、招待、除名、昇格、降格、譲渡、タグ色変更）
副マスター 招待のみ
メンバー ギルドチャット、情報閲覧

信頼の優先順位

DISTRUST（不信頼） > TRUST（信頼） > ギルドメンバー > 拒否

個人の不信頼設定はギルドの自動信頼よりも優先されます。

ギルド作成

/guild create <名前> <タグ> [説明]
例: /guild create BlueCrown BC 最強のギルド

チャットシステム

複数のチャンネルを使い分けてコミュニケーションできます。

チャットチャンネル

チャンネル プレフィックス 範囲 表示形式
ローカル （なし） 50ブロック [TAG] Player: message
グローバル ! サーバー全体 [G][TAG] Player: message
ギルド @ ギルドメンバーのみ [Guild] Player: message
ウィスパー /w <player> 1対1 [You → Player]: message

※ [TAG] はギルドタグ（ギルド所属時のみ表示、設定された色で表示）

ローマ字→ひらがな変換

ローマ字入力を自動的にひらがなに変換できます。

  • konnichiha → こんにちは
  • URL、英単語、コマンドは自動的にスキップ
  • /chat romaji on / /chat romaji off で切り替え
ヒント: ローカルチャットがデフォルトです。サーバー全体に発言するには ! を付けてください。
