Notoriety

名声システム プレイヤーガイド

名声システムとは？

Notorietyは、あなたの行動に応じて評価が変わるシステムです。善良なプレイヤーは保護され、悪行を行うプレイヤーにはペナルティが課されます。

青（Innocent）

善良なプレイヤー

  • 村人と通常価格で取引可能
  • 所有物が保護される
  • 死亡時にアイテムを保持
  • ゴーレムは中立

灰（Criminal）

犯罪者

  • 村人取引が1.5倍の価格
  • 新規設置物に所有権なし
  • 攻撃されてもペナルティなし
  • 死亡時はアイテム保持

赤（Murderer）

殺人者

  • 村人が取引を拒否
  • ゴーレムが攻撃してくる
  • 懸賞金をかけられる
  • 死亡時にアイテムを落とす

状態の変化

青から灰になる条件

悪行を行い、Alignmentが0未満になると灰色になります。

灰から青に戻る条件

時間経過（1時間ごとに+10回復）や善行でAlignmentが0以上になると青に戻ります。

赤になる条件

青プレイヤーを殺害するとPKカウントが増え、赤になります。

注意: 赤から直接青には戻れません。まず灰色期間を経る必要があります。

悪行とペナルティ

行動 ペナルティ 備考
村人を殺害（ベッドあり） -50 ベッドに紐づいている村人
他プレイヤーの所有物を盗む -50 チェストからアイテムを取る
ゴーレムを殺害 -50 周囲のゴーレムも敵対化
他プレイヤーの所有物を破壊 -10 5秒以内に戻せばキャンセル
他プレイヤーを攻撃 -1 青プレイヤーへの攻撃
村人を攻撃 -1 殺害しなくても
動物を殺す -1 村人/ゴーレムに目撃された場合
村人を殺害（ベッドなし） -1 ベッドに紐づいていない村人
ヒント: 自分で設置したブロックや、信頼しているプレイヤーが設置したブロックは自由に操作できます。

コマンド

コマンド 説明
/noty status 自分のステータスを表示
/noty status [プレイヤー] 指定プレイヤーのステータスを表示
/noty history 自分の犯罪履歴を表示
/noty trust add [プレイヤー] プレイヤーを信頼リストに追加
/noty trust remove [プレイヤー] 信頼を解除
/noty trust list 信頼リストを表示
/noty bounty set [プレイヤー] [金額] 赤プレイヤーに懸賞金をかける
/noty bounty list 懸賞金リストを表示

信頼システム

信頼したプレイヤーは、あなたの所有物にアクセスできるようになります。

  • チェストを開いてアイテムを取り出せる
  • ブロックを破壊できる
  • 攻撃しても犯罪にならない
注意: 信頼は片方向です。あなたが誰かを信頼しても、相手があなたを信頼しているとは限りません。

懸賞金システム

赤プレイヤー（殺人者）には懸賞金をかけることができます。

  • 最低金額: 100
  • 複数人が同じプレイヤーに懸賞金をかけると合算
  • 赤プレイヤーを倒すと懸賞金を獲得
  • 追加で経験値報酬も獲得（PKカウントに応じて50〜500）
